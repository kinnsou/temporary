# POS abbreviations used in the vocab game

Supported abbreviations (dot label):

- `n` = noun（名詞）
- `v` = verb（動詞）
- `adj` = adjective（形容詞）
- `adv` = adverb（副詞）
- `prep` = preposition（介系詞）
- `pron` = pronoun（代名詞）
- `conj` = conjunction（連接詞）
- `interj` = interjection（感嘆詞）
- `det` = determiner（限定詞）
- `num` = number/numeral（數詞）

How to store in `memory/vocab-history.json`:

- Single POS:
  - `"pos": "n"`
- Multi POS:
  - `"pos": ["n", "v"]`
