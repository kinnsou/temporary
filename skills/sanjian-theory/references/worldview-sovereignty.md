# 世界觀、角色與主權（整理版）

> 來源整理自《三尖理論實驗室_核心總綱_v4》《將軍論_v1_4》《三尖協議_v43》《主權火力覆蓋層_SFO_v1_0》，只保留 04-09 主幹會用到的部分。

## 1. 三層語言，不要混寫

### 1.1 事件層
市場當下做了什麼。
- 燕返
- 彈切
- 見切成功
- 三尖點

### 1.2 血統 / 角色層
這組事件後來被編進哪個世界。
- 皇帝 `t2_root`
- 將軍候選 `t2_proto` / `t2′`
- 現役將軍 `t2_work`
- Ghost / 影武者 / 諸侯

### 1.3 主權層
誰有對生死的最終解釋權。
- 哪個級別發動見切
- 哪個級別能宣告世界死亡
- 哪些低級別破位只能算戰術撤退

## 2. 角色表

### 皇帝 `t2_root`
- 世界基石
- 管世界存亡
- 只有主權級別的實體破壞，才算真正死亡
- 被破後，舊世界降級，不應硬當有效支撐

### 見習將軍 / 候選將軍 `t2_proto` / `t2′`
- 用來做第一次資格測試
- 被破 = 升官失敗
- 不等於世界死亡

### 現役將軍 `t2_work`
- 當前攻擊鏈的工作守線
- 會隨鍊式推進上移
- 被破通常代表本輪 campaign 要退，不必自動解讀成亡國

### Ghost 區
- 幾何上位於 `t2_root` 與 `t2_work` 之間
- 是「將軍陣亡、皇帝未亡」時最關鍵的壓力測試帶
- 也是殺鬼、救駕、重建與晉升最容易發生的地方

### 影武者
- 皇帝腳帶內的替身 / 第二隻腳
- 常在紙糊皇帝之後，提供 Ghost 復甦或新一輪起點線索

### 諸侯
- 皇帝與將軍都不穩時的暫時支撐 / 壓力
- 只能短打，不能美化成穩定政權

## 3. 血統論與政權分級

### 皇帝局
- 見習將軍第一次下弧試煉存活
- 皇帝腳帶完整
- 可期待較完整的 `3 + 1b + 2`
- 容許較高的延伸預期

### 諸侯局
- 見習將軍未滿月即戰死，或世界是勉強拼起來的紙皇帝
- 結構較脆
- 比較像游擊戰，不適合過度美化成長趨勢

### 第一任將軍 vs 補做將軍
- **第一任將軍**：負責授任與宣告世界進入將軍期
- **補做將軍**：立將軍失敗後的修鏈再嘗試
- 補做將軍可以不過高，但 `t2′` 必須存活

## 4. 見切螺旋，為什麼重要

> **凡下位世界的終結，常常正是上位世界的起點。**

保留這條律法，是因為 04-09 的量化主幹仍然需要它來解釋：
- 為什麼小級別失敗不一定是整體失敗
- 為什麼 Ghost 試煉後反而可能升級
- 為什麼 2H / 30m / 5m 可以在 8H 世界內反覆合法重建

### 見切必然律（保留版）
若上位級別真的在形成皇帝位或將軍位，下位級別理論上一定能找到某種可審計的見切結構。

量化上的保留結論只有兩句：
- 上位弧若成立，下位見切不應完全缺席
- 若完全找不到下位審計結構，上位結構只能暫列可疑，不能直接升格

## 5. 主權原則

> **哪個級別發動見切，哪個級別就擁有對趨勢生死的最終解釋權。**

### 三層防守
- **世界防線**：`t2_root`，管宇宙存亡
- **戰役防線**：`t2_work`，管單次 campaign 是否結束
- **微觀防線**：micro t2，管這次點火 / 加碼 / 局部觸發是否失敗

### 實務含義
- 小級別破位，不得直接推翻上位有效世界
- 只有主權級別本身明確崩壞，才承認世界死亡
- 低級別更多是在觸發 trim、demote、handoff，而不是 full liquidation

## 6. Gate 與火力覆蓋層的地位

### 三道門
- Gate 1：世界門，先確認皇帝是否仍活
- Gate 2：角色門，判斷現在是皇帝局還是諸侯局
- Gate 3：鍊式門 / 微觀確認，才談點火

### SFO / Commander 倍率
- 只保留一句：**它是風控覆蓋層，不是訊號法源。**
- 目前 skill 不把固定倍率表當核心理論，除非之後明確回頭接實盤火力模組

## 7. 04-09 的翻譯方式

- 8H = 世界信用層
- 2H = 主戰役層
- 30m = 橋接 / 執行層
- 5m = 微觀授權 / 失敗最快層

也就是說，04-09 並沒有推翻舊世界觀，而是把它翻成更適合 BattleState、Campaign/Sortie 與量化實作的語言。
