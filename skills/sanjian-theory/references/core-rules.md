# 04-09 統一主幹（精簡版）

> 本檔是 `trispike_unified_core_strategy_v0_1_2026-04-09.md` 的精簡版，並只保留舊理論中仍支撐這份主幹的部分。

## 1. 三句最高收斂

- **雙重見切在近距離做出原始 `3`。**
- **核心鍊決定這個 `3` 有沒有血統。**
- **多週期第三鍊共振，才把它正式推成可持續的 `+2`。**

## 2. 量化主體

- **8H**：世界信用層，不直接負責高頻發單
- **2H**：主戰役層，主獲利單位
- **30m**：橋接 / 拆解 / 執行層
- **5m**：微觀驗證層，Basline V1 只做斷鍊標註

目前先固定 **Baseline V1 = 8H / 2H / 30m（5m 只做斷鍊標註）**。
之後若要開 4H / 1H / 5m 等其他 profile，再另外擴充；先不要把固定比例當唯一真理，也先不要一開始就通吃所有週期。

## 3. `3 + 1b + 2` 的任務語義

- `3`：世界誕生 / 造皇帝位
- `+1b`：資格審查 / 核心鍊測試
- `+2`：通過審查後的正式推進

操作重點不是等 `+2` 全噴出才追，而是：

> **在合法 `+1b` 區，提早判斷後方 `+2` 是否已具備發動資格。**

## 4. BattleState 語言

場上任何一點都應能回答：

1. 世界在做什麼
2. 主戰役在做什麼
3. 執行層準備做什麼
4. 最小級別怎麼判失敗
5. 退出後交給誰

最低限度至少要能輸出：
- `world_state`
- `campaign_state`
- `execution_state`
- `micro_fail_rule`
- `next_handoff`

## 5. 三層止損權限

- **5m 戰術止損**：點火失敗、小見切失敗、微觀第三鍊交代就走
- **2H 戰役止損**：`+1b` 審查失敗、第二鍊斷鍊主單先退
- **8H 世界失效止損**：母世界失去信用，不再把局部失敗當合法重建

## 6. 真正保留的舊理論

以下仍是 04-09 主幹的底層法源：
- 皇帝 / 將軍 / Ghost / 諸侯
- 見切螺旋
- 主權級別與三層防守
- 血統論（皇帝局 / 諸侯局）
- 鍊式連鎖以 `t2` 存亡裁決
- 斷鍊先於命名

## 7. 被刻意降級或先拿掉的內容

以下不是現在 skill 的主幹，除非 Mark 明確要求回頭看：
- v58/v65 一類 runner 的固定門檻與倍率細節
- 舊版 Commander / Sniper 的工程話術
- 多幣種 / 資金底線 / 實盤部署細節
- 把 EMA 某個交叉關係當成最高法源的舊寫法
- 任何與 04-09 主幹無關的歷史命名堆疊

## 8. Baseline V1 的操作核心

- 先看 **8H MACD 綠柱是否收斂**，再判 8H 的 `3 / +1b / +2` 所處區段
- 再看 **2H 是否準備承接 8H 的 3 段**，並以 2H 的 `+2` 作為本版主目標
- **30m** 用燕返 / 彈切 / 見切決定是否進場
- **5m** 只用來標註 confirmed 斷鍊點與最後回穿的 break point
- **30m 斷鍊 3 次出場**（保守可 2 次），若 2H 先結束則以 2H 為準

## 9. 固定心法

- **結構優先，指標為輔**
- **先看斷鍊，再談命名**
- **下層不得篡位**
- **`+1b` 不戀戰**
- **先做 BattleState，再做進出場**
