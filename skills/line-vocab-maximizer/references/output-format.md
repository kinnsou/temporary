# LINE vocab output format

Use this compact style unless the user explicitly asks for another presentation.

## Example

```text
📚 今日單字

1. picnic
🎙️：皮克-尼克，重音在第1音節；KK [ˈpɪk.nɪk]
🏷️：野餐
例句：We will have a picnic in the park.
（我們要在公園野餐。）

2. clever
🎙️：克雷-佛，重音在第1音節；KK [ˈklev.ɚ]
🏷️：聰明的
例句：The clever boy solved the puzzle quickly.
（那個聰明的男孩很快解開了謎題。）

3. promise
🎙️：帕-米斯，重音在第1音節；KK [ˈprɑː.mɪs]
🏷️：承諾；答應
例句：I promise to finish my homework tonight.
（我答應今晚完成作業。）

🧠 複習挑戰

1. Her English is ____________.
她的英文非常優秀。
🔍 句型觀察：（優秀）放在 be 動詞後面，拿來形容 English。

2. We ____________ a small path in the forest.
我們在森林裡發現了一條小路。
🔍 句型觀察：（發現）這裡要用過去式，發現了。後面接受詞 a small path。
```

## Guardrails

- Do not write `發音：` or `意思：` in the final child-facing message
- Use emoji-only labels for those lines
- Keep the message visually light
- The review examples above are illustrative only; do not mechanically reuse their wording
- `🔍 句型觀察` must match the actual review sentence chosen that day
- `🔍 句型觀察` should point out the sentence's real verb / be 動詞 / tense / object position in plain Chinese
- Review hints should teach something useful without sounding like a grammar textbook
- Do not show the English answer directly unless the user asks for an answer-key version
